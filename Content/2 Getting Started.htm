1 Enable Dialogflow CX

  1. Open Google Cloud Console

  2. Navigate to APIs & Services → Enable APIs

  3. Search for Dialogflow API

  4. Click Enable

Enabling Dialogflow API in Google Cloud Console

 

2. Create Your First Agent

  1. Go to Dialogflow CX Console

  2. Click Create Agent

  3. Enter:

  • Agent Name

  • Default Language

  • Time Zone

  1. Click Create

Once created, the agent becomes the main container for all flows, intents, and entities.

Creating Agent Form

 

3. Console Overview

  • Left Panel: Agents, Flows, Intents, Entities

  • Center Canvas: Flow builder

  • Right Panel: Parameters, conditions, fulfilment
